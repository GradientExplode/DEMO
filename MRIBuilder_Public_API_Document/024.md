## `MRIBuilder.Variables.get_pathway` - Function

```julia
get_pathway(sequence)
```

Gets the main [`Pathway`](#MRIBuilder.Pathways.Pathway) that spins are expected to experience in the sequence.

Multiple pathways might be returned as an array or (named)tuple.
